Css 란?

css는 html 문서의 스타일을 지정하는데 사용하는 언어입니다 css는 html 요소가 어떻게 표시되어야하는지 설명함.

  • css 는 Casacading Style Sheets 의 약자입니다.
  • css 작성은 태그에서 style 속성을 사용하거나 또는 style 태그에 작성함.
  • style 태그에서는 특정 요소에 적용할 선택자(selector) 가 필요합니다. 참고로 style 태그는 head 안에 작성함.
  • Css 는 파일로 작성해서 여러 웹페이지(html파일) 에 적용되게 할 수 있습니다.
  • Css 를 저장하는 파일 확장자는 . css입니다.
  • css 프로퍼티는 글꼴 크기, 색상, 배경이미지, 테두리 등등이 있습니다. 모든 태그에 공통적인것 또는 특정태그에만 적용되는 것을 구별해서 사용합니다.
